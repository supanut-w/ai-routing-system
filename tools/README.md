# TOOLS DIRECTORY

## Concept: Atomic Tools
This directory is dedicated to housing **atomic, single-purpose tool definitions**. 

### Guidelines
1. **Modularity**: Tools should be designed to perform one specific action exceptionally well.
2. **Reusability**: Tools should be generalized enough to be used by multiple agents.
3. **Documentation**: Each tool definition must include clear input/output schemas and usage examples.
